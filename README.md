ReluzSensi - Landing Page de Brinquedos Sensoriais

Uma landing page moderna e responsiva desenvolvida para apresentar e divulgar brinquedos sensoriais voltados ao desenvolvimento infantil, estímulo cognitivo e bem-estar sensorial.

O projeto foi desenvolvido com foco em uma interface agradável, acessível e intuitiva, proporcionando uma experiência visual acolhedora para os usuários e clientes da marca.

Objetivo do Projeto

O objetivo desta aplicação é auxiliar na divulgação de brinquedos sensoriais, promovendo produtos que contribuem para:

estímulo sensorial;
desenvolvimento motor;
foco e concentração;
relaxamento e bem-estar infantil.

Além disso, o projeto busca oferecer uma presença digital profissional para a marca.

Como Executar o Projeto

Clone o repositório:

git clone https://github.com/ViniciusBernardo06/ReluzSensi-LandingPage.git

Acesse a pasta do projeto:

cd ReluzSensi-LandingPage

Abra o arquivo index.html no navegador.
